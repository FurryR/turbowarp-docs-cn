# 高清画笔

高清画笔允许画笔项目于一个更高的分辨率渲染，而不是卡在 480x360。它还会禁用编辑器中的一些坐标舍入。

:::caution
高清画笔可以显著降低性能，尤其是在全屏模式时，因为有太多像素需要被绘制了。

不是所有项目都能在启用高清画笔的情况下看起来更好。
:::
