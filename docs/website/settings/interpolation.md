# 补帧

补帧通过在两帧间进行线性插帧来让项目中的移动动画变得更加流畅。基本上就是提高帧数而不影响脚本运行速度。

:::note
译者注：似乎补帧并没有提高帧数，而只是在每一帧渲染两次，这样的说法可能是错误的。
:::

补帧不应用于：

 - 3D 项目
 - 光线追踪项目
 - 画笔项目
 - 延迟很大的项目

补帧可能无法让这些项目看起来更流畅，因为补帧无法为画笔进行插帧，而它还对性能有负面影响。

补帧可能会导致略微的 (最高 1/30 秒) 的输入延迟。

你可能会在诸如卷轴跑酷等复杂作品中看到渲染错误。
